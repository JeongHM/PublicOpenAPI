# Public Open API
---
### Open API
- data.go.kr (공공 데이터 포털)

### Spec
#### Backend
- Python 3.7
- Mysql (Maria DB) : Database
- Linux (LightSail) : Server

#### Frontend
- NodeJS
- HTML
- CSS

### Detail 
공공 데이터 포털에서 제공하는 데이터를 기반으로 새로운 API를 구성하여 제공합니다.

#### 첫 번째 미션
미세먼지 데이터를 통한 실시간 전국 도, 시 별 미세먼지 상태 제공
